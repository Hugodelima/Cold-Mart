Módulos do curso

Assim como outras linguagens, Python disponibiliza um interpretador interativo e, como o nome sugere, é interativo porque tudo que é digitado nele é analisado e interpretado no mesmo momento.

Pense numa coisa que se usa a todo instante durante o desenvolvimento de um software. Pensou? Pronto, essa coisa é variável! Entender como a linguagem lida com essas cositas é muito importante. Veja como trabalhar com elas em Python.

Nesta aula você verá como capturar dados informados pelo usuário. Em se tratando de saída de dados, isso é feito a todo momento, o resultado da linha digitada numa sessão do interpretador interativo do Python por exemplo, é uma saída!
